Prism

Created during Welcome Week as part of a team in a mini game jam, Prism is a noir-style side-scroller game. Our task was to create a game of according to a theme decided in our teams. All we knew is that we wanted Sans from Undertale in it - the rest came later. It was created over the course of four days. It can be played on itch.io using the embedded link below.


My First Week

The game was made using Unity with scripting done in C#. At the end of the week, professionals from Playground Games came in to judge all of the games that were created. Out of all of the games that were made, Prism, that game that I helped create, was voted the best even though our skills as game developers were limited at the time.
