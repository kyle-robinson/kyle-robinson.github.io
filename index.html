Few Words About Me

Hi, I'm Kyle. I'm a computer games programmer and a second year student at Staffordshire University.


C++

My primary programming language. All of my game engines and personal projects are written using C++. My aim is to develop a game in a game jam using C++.


Graphics Programming

The main graphics API that I use in my projects is OpenGL. I am currently working to create several projects using OpenGL. I also aim to learn DirectX.


Japanese

As a hobby, I teach myself Japanese, and have reached an intermediate level. I am currently planning to use this with my programming knowledge to make games programming more accessible to like-minded people in Japan.
