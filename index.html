Few Words About Me

Hi, I'm Kyle. I'm a computer games programmer and a second year student at Staffordshire University.


C++

My primary programming language. All of my game engines and personal projects are written using C++. As an ardent developer, my aim is to achieve an entry-level job with the games industry.


Graphics Programming

The main graphics API's that I use in my projects are OpenGL and DirectX 11. I am currently working to further my understanding of the DirectX 11 API and its various implementations.


Japanese

As a hobby, I teach myself Japanese, and have reached an intermediate level. I am currently planning to use this with my programming knowledge to make games programming more accessible to like-minded people in Japan. More information.
