Unity: Prism

Created during Welcome Week as part of a team in a mini game jam, Prism is a noir-style side-scroller created with Unity. Our task was to create a game according to a theme decided in our teams. After much deliberation over what features out game would include, there were numerous, but above all, the one thing that was clear to us was that Sans from Undertale would feature in it - the rest came later. Prism was created over the course of four days. It can be played on itch.io using the embedded link below. "Space Shooter" is another Unity game that I have worked on.


My First Week

Prism was made using Unity with scripting done in C#. At the end of the week, professionals from Playground Games came in to judge all of the games that were created. Out of all of the games that were made, Prism, the game that I helped create as part of my team, was voted the best - even though our skills as game developers were limited at the time. While I have been able to create more advanced games using C++ in recent times, I am still very proud of my team for being able to work together effectively to create "Prism," and make it as entertaining as it is.
